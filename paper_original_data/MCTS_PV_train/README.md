# MCTS Value & Policy Network Training

This directory contains the datasets and source code required to train the **value** and **policy** networks for Monte Carlo Tree Search (MCTS).

## Data
- `./data/filtered_MCTS_data.json`  
  Contains ~12,000 molecules with associated value (`v`) and policy/probability (`p`) labels, generated by the ReSynZ framework.

## ChemBART 
- `model4mcts.py`  
  Defines the ChemBART-based model architecture for predicting `p` and `v`.
- `train_chembart_pv.py`  
  Training pipeline/script for the ChemBART-based value & policy prediction model.

## Baseline Models
For performance comparison and ablation studies, the following baseline implementations are provided:
- `RSZ_RL/` – ReSynZ neural network baseline.
- `deepchem_gcn_baseline/` – Graph Convolutional Network (GCN) baseline (built with DeepChem).
- `rf_value_baseline/` – Random Forest baseline.
- `xgboost_baseline/` – XGBoost baseline.
