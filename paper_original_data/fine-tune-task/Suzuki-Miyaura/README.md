# Suzuki-Miyaura Reaction Yield Benchmark

This repository documents the benchmark tests for Suzuki-Miyaura C–C coupling yield prediction, including raw/processed data splits and reproduction scripts for reported results.

## Data Organization
| Directory/File | Description |
|----------------|-------------|
| `./Suzuki-Miyaura_splits/` | Raw data splits before preprocessing (train/validation/test partitions) |
| `./SM_cleaned/` | Cleaned and standardized dataset after preprocessing |
| `Yield_SM_final.ipynb` | Jupyter notebook reproducing all literature-reported results and evaluation metrics |
