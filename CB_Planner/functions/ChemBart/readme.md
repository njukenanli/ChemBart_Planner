# ChemBart: A Pre-trained Bart Model Assisting Organic Chemistry Analysis #

This is the source code of ChemBart. For our pre-trained model "Chemistry Bart" (ChemBart) Paper *ChemBart: A Pre-trained Bart Model Assisting Organic Chemistry Analysis*, see [not yet published].

## Environment ##
The original code is implemented on Linux. To set up the python environment, the following library is required.
> PyTorch (with cuda or not)
> 
> transformers
> 
> rdkit

## Demo ##

In the directory "demo", there are some main.py files to show how to run ChemBart models.

[single_step_demo.ipynb](./demo/single_step_demo.ipynb) gives examples on how to run single step models for synthesis, temperature and policy-value
